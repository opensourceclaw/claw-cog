# Claw-Cog v2.0.0 Iteration Spec

## Context

### Current State
- **pyproject.toml**: v1.5.0
- **__init__.py**: v1.8.0 (unreleased, development)
- **Architecture**: ETCLOVG (Extended Temporal Consciousness with Observation, Volition, Goal-tracking)
- **Uncommitted**: `bridge.py` (OpenClaw JSON-RPC bridge)

### v1.8.0 Features (Unreleased)
- Volition Layer (V): Goal tracking, intention selection
- Observation Layer (O): Anomaly detection, self monitoring
- ETCLOVG: O → V → C2 → C1 → C0 pipeline

## Goals

### Primary Goal
Release v2.0.0 with complete ETCLOVG architecture and production-ready features.

### Success Criteria
1. All v1.8.0 features verified and tested
2. Version synchronization (pyproject.toml, __init__.py, README.md)
3. Documentation updated
4. GitHub Release published
5. 80%+ test coverage maintained

## Constraints

### Technical Constraints
- Must maintain backward compatibility with v1.5.0 API
- Must pass all existing tests
- Must integrate with claw-mem ≥ 2.8.0 and claw-rl ≥ 2.7.0

### Project Rules (from TOOLS.md)
- All code and docs MUST be 100% English
- PUBLIC project → docs/ in .gitignore
- Release = GitHub Release (not just local tag)
