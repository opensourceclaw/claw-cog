### Task 4: neoclaw 初步集成 (P2)

**集成点**:
- CognitiveGovernance (L1-L6) + ConsciousAgent (C0-C2)
- 意识层提供"思考"能力
- 治理层提供"决策"能力

**集成方案**:

```python
# neoclaw/src/neoclaw/governance/cognitive_governance.py

class CognitiveGovernance:
    def __init__(self, ...):
        # 新增：意识层集成
        self.conscious_agent = ConsciousAgent(enable_c2=True)
    
    def process_with_consciousness(self, input, context):
        # 1. 意识层处理
        conscious_result = self.conscious_agent.process(input)
        
        # 2. 治理层决策
        governance_result = self.govern(conscious_result)
        
        # 3. 元认知反馈
        if conscious_result.confidence < 0.5:
            # 触发 L4 学习治理
            self.learning_governance.trigger_learning(conscious_result)
        
        return governance_result
```

**验收标准**:
- [ ] ConsciousAgent 可在 neoclaw 中初始化
- [ ] 意识层结果可传递到治理层
- [ ] 集成测试通过
