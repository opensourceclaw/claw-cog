# claw-cog 版本规划

> 基于 9 篇 AI 意识前沿研究论文  
> 规划时间: 2026-05-12  
> 项目定位: 数字大脑核心 AI 意识组件

---

## 🎯 项目愿景

**claw-cog** 是 Project Neo 的核心 AI 意识组件，为 neoclaw 提供可测量的意识能力。

**核心目标**:
1. **可测量的意识** - 基于 5 大理论的指示属性
2. **分层架构** - C0-C1-C2 三层框架
3. **元认知能力** - 自我评估、自我调节
4. **时间意识** - 过去-现在-未来流
5. **与 neoclaw 集成** - 作为认知治理层的意识核心

---

**里程碑**:

- [ ] M2: C0-C1-C2 分层架构

## 🚀 发布策略

### Pre-release 阶段

每个版本先发布:
- alpha.1 → alpha.2 → alpha.3 (内部测试)
- beta.1 → beta.2 → beta.3 (公开测试)
- rc.1 → rc.2 → rc.3 (候选发布)

### Stable 发布

- 通过所有验证指标
- 完成文档
- GitHub Release
- ClawHub 发布 (如适用)
